1. 프로젝트 개요

🎯 선택한 SDGs 항목 (또는 희망 진로)

SDGs 항목: 13번 - 기후 변화 대응 (Climate Action)

💡 프로젝트 주제 및 요약

최근 심각해지는 환경 문제와 기후 변화에 대응하기 위해, 일상에서 가장 쉽게 실천할 수 있는 분리수거의 중요성을 알리고자 합니다. Python의 Pygame 라이브러리를 활용하여, 청소년들이 재미있게 올바른 분리수거 방법을 학습할 수 있는 미니 게임 '분리수거의 달인'을 개발합니다. 이 게임을 통해 환경 보호에 대한 인식을 높이고 실천을 유도하는 것을 목표로 합니다.

2. 문제 분석 및 해결 계획

📌 문제 정의

많은 사람들이 분리수거의 필요성은 인지하고 있지만, 어떤 쓰레기를 어디에 버려야 하는지 정확한 방법을 모르는 경우가 많습니다. 특히 헷갈리는 품목이 많아 잘못된 배출로 인해 재활용률이 떨어지는 문제가 발생합니다. 이는 자원의 낭비와 환경오염을 심화시키는 원인이 됩니다.

🛠️ 해결 아이디어 및 방법

  • 해결 아이디어: 교육용 책자나 영상보다 더 능동적인 참여를 유도할 수 있는 '게임'이라는 매체를 활용하여 학습 효과를 높입니다.
  • 구현 방법: 하늘에서 떨어지는 여러 종류의 쓰레기 아이템을 보고, 사용자가 키보드로 바구니를 움직여 알맞은 분리수거함(플라스틱, 종이, 캔 등)에 넣는 방식으로 게임을 구현합니다.

📈 예상 동작 결과

  1. 프로그램을 실행하면 게임 제목과 '시작' 버튼이 있는 메인 화면이 나타납니다.
  2. '시작' 버튼을 누르면 게임이 시작되고, 화면 위에서 무작위로 쓰레기 아이템이 떨어집니다.
  3. 사용자는 좌우 방향키로 화면 하단의 분리수거 바구니를 움직입니다.
  4. 쓰레기가 바구니에 닿으면 점수를 얻고, 잘못된 바구니에 닿거나 바닥에 떨어지면 생명력이 감소합니다.
  5. 생명력을 모두 잃으면 게임이 종료되고 최종 점수를 보여줍니다.

3. 프로그램 설계

💻 작성한 코드

import pygame import random # 1. 게임 초기화 pygame.init() screen_width, screen_height = 800, 600 screen = pygame.display.set_mode((screen_width, screen_height)) pygame.display.set_caption("분리수거의 달인") clock = pygame.time.Clock() # 2. 게임 요소 설정 (이미지, 좌표, 속도 등) player = pygame.Rect(350, 500, 100, 50) # 플레이어 바구니 trashes = [] # 쓰레기 리스트 # 3. 게임 루프 running = True while running: # 3-1. 이벤트 처리 for event in pygame.event.get(): if event.type == pygame.QUIT: running = False # 3-2. 게임 로직 (플레이어 이동, 쓰레기 생성 및 이동, 충돌 처리) keys = pygame.key.get_pressed() if keys[pygame.K_LEFT]: player.move_ip(-5, 0) if keys[pygame.K_RIGHT]: player.move_ip(5, 0) # (이하 게임 로직 생략) # 3-3. 화면 그리기 screen.fill((255, 255, 255)) # 흰 배경 pygame.draw.rect(screen, (0, 0, 255), player) # 플레이어 그리기 # (쓰레기 그리기 로직 생략) # 3-4. 화면 업데이트 pygame.display.flip() clock.tick(60) pygame.quit()

🗺️ 코드 구조 및 흐름도

  1. 초기 설정: Pygame 라이브러리를 불러오고 게임 창의 크기, 제목 등 기본 환경을 설정합니다.
  2. 게임 요소 생성: 플레이어(바구니)와 쓰레기 객체를 생성하고, 초기 위치와 속도 등의 속성을 정의합니다.
  3. 게임 루프 진입: while 반복문을 통해 게임이 계속 실행되도록 합니다.
  4. 이벤트 처리: 사용자의 입력(키보드 누름, 창 닫기 등)을 감지하고 처리합니다.
  5. 게임 상태 업데이트: 이벤트에 따라 플레이어의 위치를 바꾸고, 새로운 쓰레기를 생성하며, 충돌 여부를 판단합니다.
  6. 화면 렌더링: 업데이트된 게임 상태를 바탕으로 모든 객체를 화면에 다시 그립니다.
  7. 화면 업데이트: display.flip()으로 실제 모니터에 그림을 표시하고, clock.tick()으로 초당 프레임 수를 조절합니다.

📚 사용한 함수, 모듈, 라이브러리

  • pygame: 게임 제작을 위한 파이썬 라이브러리. 화면 표시, 이미지 로딩, 사용자 입력 처리 등 게임의 전반적인 기능을 구현하기 위해 사용했습니다.
  • random: 파이썬 표준 라이브러리로, 쓰레기가 화면의 무작위 위치에서 나타나도록 하기 위해 random.randint() 함수를 사용했습니다.

💡 관련 교과 지식 탐구

과학 - 자유 낙하 운동: 하늘에서 떨어지는 쓰레기의 움직임은 중력의 영향을 받는 물체의 운동과 유사합니다. 게임에서는 일정한 속도로 떨어지도록 구현했지만, 가속도를 적용하여 점점 빠르게 떨어지게 만들면 실제 자유 낙하 운동( $s = v_0 t + \frac{1}{2} a t^2$ )과 유사한 물리 현상을 구현할 수 있습니다.

환경 - 자원의 순환: 분리수거는 자원을 재활용하여 환경을 보호하는 자원 순환 과정의 첫 단계입니다. 이 게임은 플라스틱, 종이, 캔 등의 재활용 과정을 간접적으로 체험하게 함으로써, 통합사회 및 과학 교과에서 배우는 환경 보호와 지속 가능한 발전의 중요성을 이해하는 데 도움을 줍니다.

4. 프로그램 구현 결과

🖼️ 주요 실행 결과 (스크린샷 첨부)

🔍 결과 해석

첫 번째 스크린샷은 게임의 시작 화면으로, 사용자가 게임에 진입할 수 있도록 안내합니다. 두 번째 스크린샷은 실제 게임 플레이 화면으로, 하늘에서 쓰레기가 떨어지고 플레이어가 바구니를 조작하는 모습을 보여줍니다. 이를 통해, 당초 계획했던 '게임을 통한 분리수거 학습'이라는 목표가 시각적으로 성공적으로 구현되었음을 확인할 수 있습니다.

5. 프로젝트 후기 및 느낀 점

🤔 진행 중 어려웠던 점과 해결 방법

어려웠던 점: 떨어지는 쓰레기와 플레이어 바구니의 충돌을 정확히 감지하는 로직을 구현하는 것이 어려웠습니다. 처음에는 좌표 계산을 직접 하려다 보니 코드가 복잡해지고 오류가 많았습니다.

해결 방법: Pygame의 내장 기능인 Rect 객체와 colliderect() 함수를 사용하는 방법을 찾아보았습니다. 각 객체를 사각형(Rect)으로 만든 뒤, 이 함수를 사용하니 두 사각형이 겹치는지 여부를 간단한 코드로 쉽게 판단할 수 있었습니다.

🌱 앞으로의 개선 방향

현재는 몇 가지 쓰레기 종류만 있지만, 앞으로는 일반쓰레기, 비닐, 유리병 등 더 다양한 품목을 추가하여 실제 분리수거와 유사한 경험을 제공하고 싶습니다. 또한, 스테이지가 올라갈수록 쓰레기가 떨어지는 속도를 높여 게임의 난이도를 조절하는 기능을 추가하면 더 재미있는 게임이 될 것 같습니다.
