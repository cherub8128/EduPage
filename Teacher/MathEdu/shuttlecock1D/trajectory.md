첨부해주신 첫 논문은 **셔틀콕을 “수직 낙하”시키고**, 공기저항 모델(무저항/선형저항/(v^2)저항)을 비교한 뒤 **(v^2)에 비례하는 저항(제곱저항)이 가장 잘 맞는다**고 결론내리는 아주 “교육용으로 깔끔한” 구조예요. 논문에서는 그 모델로부터 **종단속도 (v_T\approx 6.80,\mathrm{m/s})** 등을 추정합니다. 

* [논문 PDF 열기](sandbox:/mnt/data/peastrel1980.pdf) 

아래는 이 논문을 뼈대로 해서, **고1 수학(함수·그래프·이차함수·모델링·데이터 해석)**에 딱 맞게 설계한 **실험→가설→수학적 유도→시뮬레이션→오차 분석→확장 탐구** 수업안입니다.

## 0. 수업 전체 컨셉: 정–반–합(모델링의 표준 루트)

* **정(Thesis)**: “중력만 받는 물체” → (y(t))는 이차함수(자유낙하), 궤적은 포물선(투사체)
* **반(Antithesis)**: 셔틀콕 데이터는 포물선/자유낙하와 다르게 나온다(가속도가 일정하지 않다, 속도가 어느 순간 ‘포화’된다)
* **합(Synthesis)**: 공기저항을 추가한 모델(특히 (F\propto v^2))을 세우고, **데이터로 검증**한다

이 구조 자체가 “과학적 방법 + 수학적 모델링”의 정석입니다.

---

## 1. 수업 목표(고1 수학 단원 연결)

학생이 할 수 있게 되는 것(핵심):

1. Tracker로 **위치–시간 데이터**를 얻고, 그래프( (y-t), (v-t), (a-t) )로 운동을 해석한다.
2. 자유낙하에서 (y(t))가 **이차함수**, (v(t))가 **일차함수**임을 데이터로 확인하고 (g)를 추정한다.
3. 셔틀콕에서 **가속도가 일정하지 않음**을 확인하고, **“종단속도(terminal velocity)”** 개념을 데이터로 설명한다.
4. “가정(단순화) → 수식 모델 → 시뮬레이션 → 관측과 비교 → 오차 분석”의 과정을 보고서로 완성한다.

## 2. 차시 구성(권장 4~5차시 프로젝트형)

* **1차시**: 실험 설계(변인 설정, 통제 계획) + Tracker 사용법 연습 + “일반 물체” 자유낙하 촬영
* **2차시**: 셔틀콕 수직 낙하 촬영(같은 조건으로) + Tracker로 데이터 추출
* **3차시**: 데이터 분석(그래프/회귀/비교) + 자유낙하 식 유도(고1 수준 증명) + (g) 추정/오차
* **4차시**: 셔틀콕 모델링(제곱저항 가설) + 선형화 검증( (a) vs (v^2) ) + 종단속도 추정
* **5차시(선택)**: 시뮬레이션 제작(GeoGebra 또는 HTML+JS/Python) + 창의 확장 탐구 발표

## 3. 실험 수업 설계(과학 실험 표준 형식)

### 3.1 연구 문제(예시 문장 그대로 사용 가능)

* **RQ1**: 공기저항이 거의 없는 물체의 자유낙하에서 가속도는 정말 일정하게 (g)인가?
* **RQ2**: 셔틀콕의 수직 낙하에서 속도는 시간이 지날수록 계속 증가하는가, 아니면 일정한 값(종단속도)에 가까워지는가?
* **RQ3**: 셔틀콕의 저항력은 속도에 비례((F\propto v))할까, 속도의 제곱에 비례((F\propto v^2))할까?

### 3.2 변인 설정(필수: 조작/종속/통제)

#### 실험 A: “일반 물체” 자유낙하(공기저항 최소화)

* **조작변인(독립변인)**: 물체 종류(예: 쇠구슬/작은 고무공 등 “조밀하고 작은” 물체)
* **종속변인**: (y(t)), (v(t)), (a(t)), 낙하 시간 (t_f), 추정 중력가속도 (g_\text{exp})
* **통제변인**

  * 낙하 높이 (H) (셔틀콕 실험과 동일하게 맞추기)
  * 초기속도 (v_0=0) (손을 “옆으로 밀지 않기”, 놓는 방식 통일)
  * 촬영 프레임레이트(가능하면 60fps 이상), 카메라 거리/각도(수평 유지, 삼각대 고정)
  * 측정 스케일(줄자/스케일바), 좌표축 설정(원점/양의 방향)
  * 실내 바람/공기 흐름(선풍기 OFF, 창문 닫기)
  * 같은 장소(가능하면 계단참/복도 등)

#### 실험 B: 셔틀콕 수직 낙하

* **조작변인(독립변인)**: 물체 종류(셔틀콕 vs 일반 물체)

  * (확장 가능) 셔틀콕 종류(깃털/나일론), 초기 회전 유무, 드롭 높이
* **종속변인**: (y(t)), (v(t)), (a(t)), 추정 종단속도 (v_T), 저항계수(또는 (k)) 추정치
* **통제변인**: 위와 동일 + **셔틀콕 자세(코르크가 아래로 향하게) 최대한 일정하게**

> 논문도 “개방된 계단실(stairwell)에서 수직 낙하” 형태로 데이터를 모으고, 여러 모델 중 (v^2) 저항이 가장 잘 맞는다고 비교합니다. 

### 3.3 준비물(현실적으로 가능한 버전)

* 스마트폰(60fps 이상 권장) + 삼각대(또는 고정 지지대)
* 줄자/테이프(벽에 세로로 붙이기) + 눈금이 큰 스케일(1m 단위가 잘 보이게)

## 5. 자유낙하: 고1 수학 수준 “완전 유도/증명” (미적분 없이)

좌표: 아래쪽을 (+), 초기 (t=0)에서 (y(0)=0), (v(0)=0).

### 5.1 가정(단순화 = 가설)

* 공기저항 무시(또는 매우 작다)
* 중력가속도 (g)는 일정
* 물체는 점(크기 무시), 낙하는 수직 1차원

### 5.2 등차수열로부터 (y(t)=\frac12gt^2) 증명

시간을 (\Delta t)씩 잘라 (t_N=N\Delta t)라 하자.

* 가속도가 일정 (g)이면, 매 구간마다 속도 변화량은
  [
  \Delta v=g\Delta t
  ]
  따라서 구간 시작 속도는 등차수열:
  [
  v_n = n g\Delta t \quad(n=0,1,2,\dots)
  ]

* (n)번째 구간 ([t_n,t_{n+1}])에서의 이동거리 (\Delta y_n)는
  “평균속도 × 시간”으로 정확히:
  [
  \Delta y_n=\frac{v_n+v_{n+1}}{2}\Delta t
  = \frac{n g\Delta t + (n+1)g\Delta t}{2}\Delta t
  = \left(n+\frac12\right)g(\Delta t)^2
  ]

* 전체 이동거리:
  [
  y_N=\sum_{n=0}^{N-1}\Delta y_n
  =g(\Delta t)^2\sum_{n=0}^{N-1}\left(n+\frac12\right)
  =g(\Delta t)^2\left(\frac{N(N-1)}{2}+\frac{N}{2}\right)
  =\frac{g}{2}N^2(\Delta t)^2
  ]
  그런데 (t_N=N\Delta t)이므로
  [
  y(t)=\frac12gt^2
  ]
  증명 끝.

일반 초기조건((y_0, v_0))까지 포함하면:
[
y(t)=y_0+v_0t+\frac12gt^2,\qquad v(t)=v_0+gt
]

### 5.3 실측 비교(학생 활동)

* Tracker에서 (y(t)) 데이터를 **이차함수로 피팅**:
  [
  y(t)\approx at^2+bt+c
  ]
  이때 (아래쪽 (+)라면) (g_\text{exp}\approx 2a).

* **오차(%)**:
  [
  \text{오차율}=\left|\frac{g_\text{exp}-9.80}{9.80}\right|\times 100%
  ]

* 오차 원인 토의(필수):

  * 줄자 평면 불일치(패럴럭스), 클릭 추적 오차, 프레임레이트 오차
  * 실제로는 공기저항이 0이 아님(특히 큰 공일수록)
  * 초기속도 0이 완벽하지 않음(손이 밀림)

## 6. 셔틀콕: 논문 방식 따라가기(고1 수준 모델링 + 심화 유도 옵션)

논문은 여러 저항 모델 중 **저항력이 순간 속도의 제곱에 비례**한다고 두는 모델이 가장 잘 맞았다고 보고합니다. 
그 핵심 가정은 이렇게 요약됩니다:

### 6.1 가설(단순화) 세우기

* 운동은 수직 1차원
* 중력 (mg)는 일정(아래 방향)
* 저항력은 위 방향, 크기는 (v^2)에 비례:
  [
  F_\text{res}=k m v^2 \quad(\text{위 방향})
  ]
  (논문은 같은 형태로 두고 전개합니다. )

### 6.2 뉴턴 법칙 → “고1에서 바로 쓰는 핵심식”

아래쪽 (+)로 두면 힘의 합:
[
m a = mg - k m v^2
]
따라서
[
a = g - k v^2
]
이 한 줄이 고1 친화적입니다. 이유:

* (a)와 (v^2)의 관계가 **일차함수(직선)**이기 때문!

### 6.3 종단속도(terminal velocity) 도출

“속도가 더는 증가하지 않을 때”는 (a=0):
[
0=g-kv_T^2 \Rightarrow v_T=\sqrt{\frac{g}{k}}
]
이제 학생들은 **데이터로 (v_T)**를 찾아낼 수 있습니다.

## 7. 셔틀콕 데이터로 모델 검증(핵심 활동: 선형화 플롯)

### 7.1 Tracker에서 얻을 것

* (v(t)) (수직속도 (v_y(t)))
* (a(t)) (수직가속도 (a_y(t)))

### 7.2 모델이 맞다면 나타나야 하는 패턴

[
a = g - k v^2
]
즉, **가로축 (v^2), 세로축 (a)**로 산점도를 찍으면 **거의 직선**이 나와야 합니다.

* 절편 (\approx g)
* 기울기 (\approx -k)
* (v_T=\sqrt{\frac{g}{k}})

이건 “미분방정식 풀이” 없이도 **모델의 형태 자체를 실험으로 검증**하는 방식이라, 고1 수학(직선의 방정식/함수해석)과 매우 잘 맞습니다.

### 7.3 실습 루틴(학생용)

1. Tracker에서 (v_y, a_y) 데이터 추출
2. 새 열로 (v_y^2) 계산
3. (a_y) vs (v_y^2) 산점도 작성
4. 직선 피팅: (a = \alpha + \beta v^2)
5. (k=-\beta), (v_T=\sqrt{\alpha/(-\beta)}) 계산
6. 논문 값(약 (6.8,\mathrm{m/s}))과 비교(“같아야 한다”가 아니라, **왜 다를 수 있는지**까지 분석)

## 9. “포물선 운동” 가설 설정 단계(수학적으로 충실하게)

여기서부터는 수직낙하에서 한 단계 확장해 **투사체 운동(포물선)**을 다룹니다. (실험 확장으로 공/셔틀콕을 옆으로 던져 촬영해도 좋고, 이론+시뮬레이션만으로도 가능)

### 9.1 포물선 운동의 단순화 조건(학생이 직접 체크하도록)

* [ ] 공기저항/바람 무시
* [ ] 중력은 일정하고 아래로만 작용
* [ ] 물체는 점(크기/회전 무시)
* [ ] 지면은 평평, 회전좌표 효과 무시
* [ ] 초기속도 (\vec v_0)만 주어지면 이후는 법칙으로 결정

이 조건 아래에서:
[
x(t)=x_0+v_0\cos\theta;t,\qquad
y(t)=y_0+v_0\sin\theta;t-\frac12gt^2
]
시간 (t)를 소거하면:
[
y(x)=y_0+(x-x_0)\tan\theta-\frac{g}{2v_0^2\cos^2\theta}(x-x_0)^2
]
즉 **이차함수(포물선)**.

### 9.2 셔틀콕은 왜 포물선이 깨지나(가설의 수정)

* 공기저항이 커서 (x)방향 속도도 감소
* (y)방향 가속도도 일정하지 않음(위에서 본 (a=g-kv^2) 류)
* 따라서 궤적은 더 이상 (y(x))가 “깔끔한 이차함수”가 아니다

이게 바로 “정→반→합”의 자연스러운 전개입니다.

## 10. (심화/교사용) 논문식 “완전 유도”: (v(t)=v_T\tanh(\cdot))까지

논문은 제곱저항 모델을 놓고 해를 적분해 닫힌형으로 표현합니다. 
(학생 필수는 아니고, **수학 동아리/심화 보고서**에 넣기 좋습니다.)

[
\frac{dv}{dt}=g-kv^2
]
여기서 (v_T=\sqrt{\frac{g}{k}})를 쓰면
[
\frac{dv}{dt}=g\left(1-\frac{v^2}{v_T^2}\right)
]
변수분리:
[
\frac{dv}{1-(v^2/v_T^2)}=g,dt
]
(u=v/v_T)로 두면
[
\int \frac{du}{1-u^2} = \frac{g}{v_T}\int dt
]
좌변은 (\operatorname{artanh}(u))가 되고,
[
\operatorname{artanh}(u)=\frac{g}{v_T}t
\Rightarrow u=\tanh\left(\frac{g}{v_T}t\right)
]
따라서
[
v(t)=v_T\tanh\left(\frac{g}{v_T}t\right)
]
그리고 (y'(t)=v(t))를 적분하면
[
y(t)=\frac{v_T^2}{g}\ln\cosh\left(\frac{g}{v_T}t\right)
]
이 형태의 식을 논문이 사용합니다.

## 11. 관찰 vs 이론 비교, 오차 원인(자유낙하/셔틀콕 공통)

### 11.1 자유낙하에서 대표 오차

* 공기저항 완전 무시 불가(특히 큰 공)
* 프레임레이트/셔터로 인한 시간·좌표 오차
* 스케일이 낙하평면과 어긋나면 길이 보정이 틀어짐(가장 흔한 실수)
* Tracker 미분(속도/가속도) 잡음

### 11.2 셔틀콕에서 추가 오차(“현실이 더 괴물”)

* 낙하 중 자세 변화(흔들림/회전) → 저항계수가 시간에 따라 변함
* 셔틀콕 종류·마모·깃 변형에 따른 차이
* 계단실 공기 흐름(상승기류) 영향
* 바닥 근처 난류/벽면 효과

오차 분석을 “변명”이 아니라 “모델 개선의 실마리”로 다루게 하는 게 포인트입니다.

## 12. 확장·창의 탐구 아이디어(논문 ‘논의’에서 출발해 학생이 바꿔보기)

아래는 보고서/발표 주제로 바로 쓰기 좋게 구성했습니다.

1. **변인 추가(실험 확장)**

* 셔틀콕 종류(깃털 vs 나일론)별 (v_T) 비교
* 셔틀콕에 테이프를 조금 붙여 질량 변화 → (v_T)가 어떻게 변하는가?
* “스핀을 주고 떨어뜨리면” 데이터가 어떻게 달라지는가?
* 같은 높이에서 “코르크 위/아래”로 시작하면 초기 구간이 어떻게 달라지는가?

2. **문제 바꿔 생각하기(수학적 질문 만들기)**

* (최적화) “셔틀콕이 특정 시간 안에 가장 느리게 떨어지게” 하려면 어떤 변인을 조절해야 할까?
* (역문제) 측정한 (v_T)로부터 (k)를 추정했을 때, 그 값이 “환경(공기 밀도)”에 따라 어떻게 바뀔까?

3. **실생활 연결(배드민턴 전술과 물리)**

* 왜 배드민턴은 공 대신 셔틀을 쓰나?
  → 종단속도/강한 항력이 경기 공간과 반응 시간을 “인간 친화적”으로 만든다.
* 왜 클리어(clear)는 길게 가고, 드롭(drop)은 갑자기 떨어지는가?
  → 속도 영역에 따라 저항이 운동을 지배하는 방식이 달라짐.

4. **모델 경쟁(정–반–합을 한 번 더)**

* 선형저항 vs 제곱저항 vs “구간별 모델(저속은 선형, 고속은 제곱)”
  어떤 모델이 데이터에 더 설득력 있게 맞는지 **R²/잔차 그래프**로 경쟁하게 하기.
