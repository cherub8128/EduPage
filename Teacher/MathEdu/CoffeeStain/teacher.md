# (교사용) “커피 얼룩(커피 링 효과)” 수업 지도안 — 실험→관찰→추상화→시뮬→검증→수식화→확장질문

> 핵심 한 줄: **접촉선(contact line)이 고정(pinning)된 액적은 가장자리 증발이 더 커서, 그 손실을 메우려는 “바깥쪽 모세관 유동”이 생기고, 입자(커피/미세분말)가 거의 전부 가장자리로 운반되어 링을 만든다.** 
> (확장) 증발이 아주 빠르면 입자가 표면에 “붙잡혀(surface capture)” 가장자리로 못 가서 **더 균일한 침전**으로 바뀔 수 있다.

## 0. 수업 목표(통계물리+수학 연결)

### 개념 목표(통계물리)

* 비평형(비정상) 과정에서의 **물질수송(유동-확산-포획)**이 패턴(링/균일)을 만든다는 것을 이해
* “미시(입자) → 거시(침전 패턴)” 연결: **보존법칙 + 경계조건(핀닝) + 스케일링 법칙**

### 수학 목표(고등 확률·통계 / 미적분)

* (미적분) 연속체 관점: **증발 플럭스 (J(r))** → **연속방정식** → **평균 방사속도 (\bar v(r))** 도출
* (확통) 관측 데이터로 **로그–로그 회귀**(멱법칙 지수 추정), 잔차/불확실성 논의
* (확통) “입자 출발 위치 = 확률변수”로 두고, **도착시간 분포**와 **누적 링 질량 (M(t))**를 분포함수로 해석

## 3. 이론(교사용 상세 수학 전개 + ‘증명’ 형태)

### 3.1 증발 플럭스 (J(r)): 왜 가장자리에서 커지나?

Deegan은 공기 중 수증기 농도 (f)가 (준정상) **라플라스 방정식**
[
\nabla^2 f = 0
]
을 만족하고, 표면에서의 기울기가 증발 플럭스를 준다고 둔다:
[
J(r)= -D,\frac{\partial f}{\partial n}
]
(여기서 (D)는 공기 중 확산계수). 
그리고 기하학적으로 “쐐기(wedge)” 경계 때문에 가장자리에서 기울기가 발산하여
[
J(r)\sim (R-r)^{-\lambda},\qquad
\lambda=\frac{\pi-2\theta_c}{2\pi-2\theta_c}
]
를 준다. 

> **수업용 해석:** 기판이 “수증기 확산의 반사면”처럼 작동해 가장자리 근처가 뾰족한 경계조건이 되고, 그 결과 플럭스가 커진다(전기장-도체 모서리에서 발산하는 것과 유사).

---

### 3.2 핀닝 + 보존법칙 ⇒ 바깥쪽 평균 유속 (\bar v(r))

핵심 논리(Deegan의 그림 2 설명): **반지름이 줄어들 수 없으니**, 가장자리 증발로 빠져나간 부피를 내부에서 바깥으로 흘려서 보충해야 한다. 

축대칭 얇은 액적에서 “반지름 (r) 바깥쪽((r)~(R))에서 증발한 양”은
[
\dot V_{\text{evap}}(r)=\int_r^R 2\pi r',J(r'),dr'
]
이 부피가 **반지름 (r)**를 통과하는 방사 유량으로 보충되므로
[
2\pi r, h(r),\bar v(r)=\int_r^R 2\pi r',J(r'),dr'.
]
따라서
[
\boxed{;\bar v(r)=\frac{1}{r,h(r)}\int_r^R r'J(r'),dr';}
]
(여기서 (h(r))는 그 순간의 높이 프로필).

#### (근사) 가장자리 근처 스케일링으로 (\bar v\sim (R-r)^{-\lambda}) “증명”

가장자리 좌표 (x=R-r) (작은 (x))를 두고,

* 증발: (J \sim x^{-\lambda})
* 높이: 접촉각이 (\theta_c)이면 가장자리에서 (h\sim \theta_c x)

그러면 적분은 ( \int_0^x s^{-\lambda}ds \sim x^{1-\lambda}) (단, (\lambda<1)) 이고,
[
\bar v \sim \frac{x^{1-\lambda}}{x} = x^{-\lambda}.
]
즉,
[
\boxed{;\bar v(r)\sim (R-r)^{-\lambda};}
]
가 된다. (Deegan이 “발산 플럭스를 대체하려면 속도도 발산”이라고 요약한 내용과 동일한 결론.) 

---

### 3.3 링 질량의 초기 성장: (M(t)\sim t^{\frac{2}{1+\lambda}}) (전개 포함)

Deegan의 핵심 결과는 다음과 같은 멱법칙이다. 
여기서는 “왜 지수가 (2/(1+\lambda))인가”를 교사용으로 깔끔히 전개한다.

#### Step 1) 가장자리 근처 “초기 질량” 스케일

입자 농도가 희석이고(초기 균일), 가장자리 쐐기 부피는

* 둘레 (2\pi R)
* 단면에서 높이 (h\sim \theta_c x)
* 두께 (dx)

따라서 (x)까지(가장자리로부터 거리 (x) 이내)에 들어있는 **용질(입자) 질량**은
[
M(x,0)\propto \int_0^x 2\pi R,(\theta_c s),ds \propto x^2.
]
즉,
[
\boxed{M(R-x,0)\sim x^2.}
]
(Deegan이 “(M(r,0)\sim (R-r)^2)”라고 쓴 바로 그 스케일.) 

#### Step 2) 도달 시간 (t(x)) 스케일

입자(또는 유체 요소)가 가장자리로 이동하는 데 걸리는 시간:
[
t=\int_{r_t}^{R}\frac{dr}{\bar v(r)}.
]
가장자리 변수 (x=R-r), (\bar v\sim x^{-\lambda})이면
[
t \sim \int_0^{x_t}\frac{dx}{x^{-\lambda}}=\int_0^{x_t}x^{\lambda},dx
\sim x_t^{1+\lambda}.
]
따라서
[
\boxed{x_t\sim t^{\frac{1}{1+\lambda}}.}
]
(Deegan의 식 (1)과 동일한 스케일 관계.) 

#### Step 3) 누적 링 질량 (M(t))

시간 (t)까지 링으로 빨려 들어간 질량은 “초기에 (x_t) 안에 있던 질량”과 스케일이 같으므로
[
M(t)\sim M(x_t,0)\sim x_t^2 \sim \left(t^{\frac{1}{1+\lambda}}\right)^2
= t^{\frac{2}{1+\lambda}}.
]
즉
[
\boxed{;M(t)\sim t^{\frac{2}{1+\lambda}};}
]
이 된다. 

#### 숫자 체크(수업에서 “오!” 하는 포인트)

접촉각이 작아 (\lambda\to 1/2)이면
[
\frac{2}{1+\lambda}=\frac{2}{1.5}=\frac{4}{3}\approx 1.33.
]
Deegan 실험에서는 (p\approx 1.3\pm 0.1) 수준의 지수를 보고한다. 
→ **“스케일링으로 지수 예측 → 로그–로그 그래프로 검증”**이 딱 고등 미적분+통계 프로젝트가 된다.

## 4. 확률·통계로 “분포함수”를 진짜로 다루는 방법(교사용)

### 4.1 입자 출발 위치를 확률변수로 모델링

가장자리 거리 (X)를 확률변수로 두고(초기 용질이 부피에 균일),
[
M(X<x)\propto x^2
\Rightarrow F_X(x)=P(X\le x)=\left(\frac{x}{x_{\max}}\right)^2.
]
따라서 확률밀도함수(pdf)
[
f_X(x)=F_X'(x)=\frac{2x}{x_{\max}^2}.
]

### 4.2 도착시간 (T)의 분포

스케일링 관계 (T\propto X^{1+\lambda}) (단조 증가)라면 (X=(T/c)^{1/(1+\lambda)}).
변수변환으로
[
F_T(t)=P(T\le t)=P!\left(X\le (t/c)^{1/(1+\lambda)}\right)
=\left(\frac{(t/c)^{1/(1+\lambda)}}{x_{\max}}\right)^2
\propto t^{\frac{2}{1+\lambda}}.
]
즉 누적 링 질량이 사실상 (F_T(t)) (누적분포함수)와 동형이다.
→ **“멱법칙 = 분포함수의 꼬리/스케일링”**을 분포 관점으로 깔끔히 연결 가능.

### 4.3 데이터 처리(고등 통계)

* 타임랩스에서 일정 시간 간격으로 링 영역 픽셀 강도 합 (I(t))를 (M(t)) 대용량으로 사용
* (\log M)–(\log t) 선형회귀로 기울기 (p) 추정
* 신뢰구간(표준오차)까지 하면 “연구 감성” 완성

## 5. 시뮬레이션(수업 수준별 3단)

### 레벨 A: “지수만 맞추는” 몬테카를로(가장 쉽고 강력)

* (X\sim f_X(x)=2x/x_{\max}^2) 샘플링
* (T=cX^{1+\lambda})로 변환
* 누적 (M(t)\propto #{T\le t})를 그려 (\log)-(\log) 기울기 확인

```python
import numpy as np

N = 20000
lam = 0.5          # λ
xmax = 1.0
c = 1.0

u = np.random.rand(N)
X = xmax*np.sqrt(u)                  # F_X(x)=(x/xmax)^2 역변환
T = c*(X**(1+lam))

t_grid = np.logspace(-4, 0, 60)
M = np.array([(T <= t).mean() for t in t_grid])  # 누적비율 ~ M(t)/M_total

# log-log 기울기(중간 구간 선형회귀)
mask = (t_grid>1e-3) & (t_grid<3e-1)
p = np.polyfit(np.log(t_grid[mask]), np.log(M[mask]), 1)[0]
print("estimated slope p ~", p)
```

* 기대: (p\approx 2/(1+\lambda)). (예: (\lambda=0.5\Rightarrow p\approx1.33))

### 레벨 B: 증발 모델을 바꿔 링 패턴 비교(“상수 플럭스 vs 가장자리 강화”)

PRF(2024)는 문헌에서 흔히 쓰는 국소 증발 플럭스 모델 2가지를 정리한다. 

* **Kinetic:** (J_{\text{evap}}(r)=J_k) (거의 균일)
* **Diffusive:** (J_{\text{evap}}(r)\propto \dfrac{1}{\sqrt{1-(r/R)^2}}) (가장자리 강화)

수업 아이디어:

* 레벨 A의 (\lambda)를 “실효 파라미터”로 두고, 기판/조건에 따라 (\lambda)가 커지거나 작아지는 시나리오로 비교
* 또는 연속식으로 (\bar v(r))를 수치적분해 입자를 이동(교사용 심화)

### 레벨 C: “표면 포획(surface capture)”로 링 억제까지 재현(심화/탐구반)

PRF(2024)의 관점:

* 증발이 매우 빠르면, 입자가 **확산으로 도망치기 전에** 표면에 붙잡힌다. 
* 이때 침전은 링이 아니라 **더 균일**해질 수 있다. 

핵심 무차원수(설명용):

* 수직 이류 시간 (t_a^*=\rho^*R^*/J^*)
* 수직 분산(확산) 시간 (t_d^*=2R^{*2}/D^*) 
* 비율이 크면(대략 (t_d^*/t_a^*\sim 2Pe) 큼) 표면 포획이 강해짐.
  PRF는 식 전개 속에서 (2Pe)가 수직 확산항에 들어가는 형태를 보인다. 

수업 구현(간이 규칙):

* 각 입자에 대해 “표면 포획 확률” (p_{\text{cap}}=1-\exp(-\alpha\cdot 2Pe)) 같은 경험식을 두고,
* 포획되면 더 이상 가장자리로 안 가고 “그 위치에 침전”되게 하면
  링→균일 전이가 정성적으로 재현된다(실험의 선풍기/가열 조건과 연결).

## 7. 예상 오개념 & 교사용 코칭 포인트

* **“가장자리로 ‘끌어당기는 힘’이 있어서 링이 된다”**
  → 힘이 아니라 **보존법칙(부피 손실 보충) + 경계조건(핀닝)**이 만든 유동이다. 
* **“확산이 링을 만든다”**
  → 링 형성의 주역은 **이류(유동) + 정체/잼밍**이고, PRF도 커피링이 “이류-확산 균형”이 아니라 잼밍으로 생긴다는 점을 강조한다. 
* **“항상 링만 생긴다”**
  → 증발이 빠르면 **surface capture**로 링이 약해지고 더 균일해질 수 있다.

## 8. 평가(루브릭 예시)

* A(우수):

  * 핀닝/증발 플럭스/보존법칙을 연결해 링 형성을 설명
  * (\lambda)–(p) 관계 (p=2/(1+\lambda))를 스스로 전개
  * 로그–로그 회귀로 (p) 추정, 불확실성/구간 선택을 논리적으로 정당화
* B(보통):

  * 실험 결과를 정량화하고 멱법칙 적합을 수행하나, 이론 전개 일부가 단편적
* C(기초):

  * 관찰 중심 서술, 정량화/모델링 연결이 약함

## 9. 확장 질문(탐구를 “논문 느낌”으로 올리는 질문들)

1. **기판이 거칠수록 핀닝이 강해져 링이 더 강해질까?** (핀닝 조건 확인) 
2. **선풍기/가열로 증발률을 키우면 링이 약해질 수도 있는데, 그 경계는?**
   → (t_a^*) vs (t_d^*) (즉 (2Pe)) 관점으로 실험 설계 
3. **증발 플럭스가 “거의 상수”인 상황을 만들면 침전이 정말 균일해지나?**
   → PRF는 표면층이 형성되면 침전이 증발 플럭스 분포를 “따라가며”, 상수 플럭스일 때 가장 균일하다고 말한다. 
4. **마랑고니(표면장력 구배)로 링을 깨뜨릴 수 있는가?**
   → 세제/알코올 미량 첨가, 패턴 변화 기록(정성+정량)

### 마무리(교사용 한 문장 요약)

* Deegan(1997): **핀닝된 액적**에서 **가장자리 강화된 증발 플럭스**가 **외향 유동**을 만들고, 그 결과 링 질량이 초기에는 (M(t)\sim t^{2/(1+\lambda)})로 자란다. 
* Coombs–Sprittles–Chubynsky(2024): 증발이 매우 빠르면 **surface capture**가 링을 억제해 **더 균일한 침전**으로 전이할 수 있고, 그 경계는 **수직 이류/확산 시간비(= (2Pe))**로 해석된다.
