STEP 4 수식화: 일반식 · 무차원화 · 다른 기하로의 확장

관찰과 시뮬레이션을 수식으로 정리하고, 무차원화와 다른 기하(원통, 정육면체)로의 확장 아이디어까지 연결합니다.

[1D 포물선 모델 - 직사각형 수조] 자유수면 높이: \[ z(r) = H - \frac{\omega^2}{2g}\left(\frac{L^2}{12} - r^2\right) \] 이를 정리하면: \[ z(r) = \frac{\omega^2}{2g} r^2 + \left(H - \frac{\omega^2 L^2}{24 g}\right) \] → 포물선의 계수는 \(A = \dfrac{\omega^2}{2g}\) 꼭짓점 높이: \[ z_v = H - \frac{\omega^2 L^2}{24 g} \] 임계 각속도 (꼭짓점이 바닥에 닿을 때): \[ z_v = 0 \quad \Rightarrow \quad \omega_c = \sqrt{\frac{24 g H}{L^2}} \] [무차원화] 길이 스케일 \(R = L/2\) 로 정규화: \[ X = \omega \sqrt{\frac{R}{g}}, \quad H^* = \frac{H}{R}, \quad Z = \frac{z}{R}, \quad R^* = \frac{r}{R} \] 그러면 무차원 자유수면 방정식: \[ Z(R^*) = H^* - \frac{X^2}{6}\left(1 - (R^*)^2\right) \] 임계 조건 (무차원): \[ X_c^2 = 6 H^* \] → 여러 크기의 물통을 하나의 무차원 곡선으로 통합 가능
무차원화의 장점
  • 일반성: 다양한 크기의 실험을 하나의 곡선으로 표현
  • 데이터 분석: \(X\) vs \(Z\) 그래프로 모든 데이터를 통합
  • 임계점 판단: \(X < \sqrt{6H^*}\) vs \(X > \sqrt{6H^*}\) 로 간단히 구분
다른 기하학적 형태로의 확장

🔵 원통 수조

반지름 \(R\) 인 원통 수조에서는:

\[ z(r) = \frac{\omega^2}{2g} r^2 + C, \quad 0 \le r \le R \] 부피 보존 조건: \[ \int_0^R z(r) \cdot 2\pi r\, dr = \pi R^2 H \] 이를 풀면: \[ C = H - \frac{\omega^2 R^2}{4g} \] 임계 각속도: \[ \omega_c = \sqrt{\frac{4gH}{R^2}} \]

🟩 정육면체 수조

한 변의 길이가 \(a\) 인 정육면체 수조에서는 2차원 포물면:

\[ z(x,y) = \frac{\omega^2}{2g}(x^2 + y^2) + C \] 마른 영역이 생기면 자유수면이 벽에서 끊기고, "마른 반지름"에 대해 3차 또는 4차 방정식을 풀어야 함. → Menker & Herczyński (2022) 논문에서 자세히 다룸

🔍 비교 분석 프로젝트

  • 직사각형 vs 원통 vs 정육면체에서 \(\omega_c\) 의 스케일링 비교
  • 각 형태에서 마른 영역의 크기 비교
  • 부피 보존 조건이 형상에 미치는 영향 분석

추천 심화 연구 주제
1. 중력가속도 측정

다양한 ω에서 포물선 곡률 A를 측정하고, ω² vs A 그래프의 기울기를 통해 g를 역추정. 실제 9.8 m/s²와의 오차 분석 및 오차 원인 토론.

2. 점성 효과 연구

물 대신 꿀, 식용유 등 점성이 다른 유체를 사용하여 정상 상태 도달 시간(이완 시간) 측정. 점성 계수와의 관계 탐구.

3. 형상별 임계 각속도 비교

직사각형, 원통, 정육면체 수조에서 마른 영역이 생기는 임계 각속도 ω_c를 실험 또는 시뮬레이션으로 비교. 부피와 형상의 관계 분석.

4. 이론 vs 물리엔진 편차 분석

Cannon.js 입자 유체의 표면 높이를 샘플링하여 이론 포물선과의 잔차(residual)를 계산. 입자 수, 반지름 등의 영향 연구.

5. 표면 장력·접촉각 효과

소형 용기나 수은 등에서는 표면 장력이 무시할 수 없음. 이론 모델에 Young-Laplace 방정식 추가하여 보정.

6. 비균일 회전 (시간 변화)

ω가 시간에 따라 증가/감소하는 경우의 과도 상태(transient) 연구. 에너지 소산률, 진동 모드 관찰.
