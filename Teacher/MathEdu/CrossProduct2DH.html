문제

1) 왜 \( {}^{\perp}\!\cdot \) (perp·dot)인가?

  • 2D 문제에서 3D 외적을 도입하면 \(k\)축과 오른손규칙 해석이 개입한다.
  • 90° 회전 연산자 \(E\)와 \( {}^{\perp}\!\cdot \) 연산으로 2D 내부에서 면적·토크·회전을 일관되게 표현할 수 있다.
  • 정의: \( \mathbf a\,{}^{\perp}\!\cdot\,\mathbf b := a_x b_y - a_y b_x \).
개념

2) 핵심 개념 — 이해 가이드

2D 직교 연산자 (perp)
\( E=\begin{pmatrix}0&-1\\[2pt]1&0\end{pmatrix}, \quad \tilde{\mathbf a}=E\mathbf a=(-a_y,a_x) \)
  • \(E^T=-E,\; E^2=-I\), \(\tilde{\tilde{\mathbf a}}=-\mathbf a\).
  • 복소수 \(i\)와 대응(90° 회전).
\( {}^{\perp}\!\cdot \) (부호가 있는 면적)
\( \mathbf a\,{}^{\perp}\!\cdot\,\mathbf b = a_x b_y - a_y b_x = \det\!\begin{pmatrix}a_x&a_y\\b_x&b_y\end{pmatrix} \)
  • 평행사변형의 부호가 있는 면적(반시계 +, 시계 −).
  • 3D 외적의 \(k\)-성분과 동일.
회전 분해
\( \mathbf b=\cos\varphi\,\mathbf a+\sin\varphi\,\tilde{\mathbf a} \)
  • \( \cos\varphi=\dfrac{\mathbf a\cdot\mathbf b}{\|\mathbf a\|\,\|\mathbf b\|} \)
  • \( \sin\varphi=\dfrac{\mathbf a\,{}^{\perp}\!\cdot\,\mathbf b}{\|\mathbf a\|\,\|\mathbf b\|} \)
  • 행렬 \(R_\varphi\), 복소수 \(e^{i\varphi}\)와 동치.
2D 토크(모멘트)
\( \tau=\mathbf r\,{}^{\perp}\!\cdot\,\mathbf F \)
  • 스칼라 결과(부호로 방향 표현).
  • 레버암 × 힘.
정리

3) 주요 정리 · 동치

R(φ)와 perp 분해
\( R_\varphi=\cos\varphi\,I+\sin\varphi\,E \), \quad \( \mathbf b=R_\varphi\mathbf a \)
  • \(e^{\varphi E}=\cos\varphi\,I+\sin\varphi\,E\), \(E^2=-I\).
슈레이스(다각형 면적)
\( A=\frac{1}{2}\sum_{i=1}^n \mathbf v_i\,{}^{\perp}\!\cdot\,\mathbf v_{i+1} \)
  • 정점 반시계면 \(A>0\).
so(2), 복소수 대응
  • \(\mathfrak{so}(2)=\text{span}\{E\}\), \([E,E]=0\).
  • \(E \leftrightarrow i\), \( \mathbf a\,{}^{\perp}\!\cdot\,\mathbf b \leftrightarrow \Im(\overline{z_a} z_b)\).
주의

4) 오해/실수

  • 2D에서 3D 벡터 토크 사용(방향을 \(k\)축으로 고정).
  • 면적을 항상 양수로 처리해 방향(부호) 정보 소실.
  • 좌표계 변경 시 부호 뒤섞임.
예제

5) 예제 · 패턴 · 체크

E1. 토크 계산
  • \(\mathbf r=(3,1),\ \mathbf F=(2,4)\)
  • \(\tau=\mathbf r\,{}^{\perp}\!\cdot\,\mathbf F=(-1,3)\cdot(2,4)=10>0\) (반시계)
E2. 회전식
  • \(\mathbf a=(2,0),\ \varphi=30^\circ\Rightarrow \mathbf b=(\sqrt{3},1)\).
체크리스트
  • \(\mathbf a\,{}^{\perp}\!\cdot\,\mathbf b\) 부호와 실제 회전 방향의 일치.
  • 정점 순서(시계/반시계)와 면적 부호의 일관성.
  • 좌표 변환 후 결과 불변성.
증명

6) 증명 스케치

▶ (A) \(e^{\varphi E}=\cos\varphi\,I+\sin\varphi\,E\)
  • \(E^{2k}=(-1)^k I,\ E^{2k+1}=(-1)^k E\) → 테일러 전개로 귀결.
▶ (B) \(\det(\mathbf a,\mathbf b)=\mathbf a\,{}^{\perp}\!\cdot\,\mathbf b\)
  • 선형·교대성·기저 일치.
▶ (C) 토크의 스칼라 표현
  • \(\tau=\|\mathbf r\|\|\mathbf F\|\sin\theta = \mathbf r\,{}^{\perp}\!\cdot\,\mathbf F\).
구현

7) 구현 가이드 — Python / MATLAB

Python
import math, matplotlib.pyplot as plt
def perp(a): x,y=a; return (-y, x)
def dot(a,b): return a[0]*b[0]+a[1]*b[1]
def pdot(a,b): return a[0]*b[1]-a[1]*b[0]
a=(2,0); b=(1,3)
print("a·b =", dot(a,b))
print("^{⊥}· =", pdot(a,b))

# mini-plot
A=[0,a[0]]; Ay=[0,a[1]]
B=[0,b[0]]; By=[0,b[1]]
plt.figure(); plt.axhline(0,color='k',linewidth=.5); plt.axvline(0,color='k',linewidth=.5)
plt.plot(A,Ay, label='a'); plt.plot(B,By, label='b')
plt.axis('equal'); plt.legend(); plt.show()
MATLAB / Octave
perp = @(a) [-a(2), a(1)];
dot2 = @(a,b) a(1)*b(1) + a(2)*b(2);
pdot = @(a,b) a(1)*b(2) - a(2)*b(1);
a=[2,0]; b=[1,3];
disp(['a·b = ' num2str(dot2(a,b))])
disp(['^{⊥}· = ' num2str(pdot(a,b))])

plot([0 a(1)], [0 a(2)], 'LineWidth',2); hold on;
plot([0 b(1)], [0 b(2)], 'LineWidth',2);
plot(xlim, [0 0], 'k-'); plot([0 0], ylim, 'k-');
axis equal; legend('a','b'); hold off;
연습

8) 연습 문제

  1. \(\mathbf a=(1,2), \mathbf b=(3,1)\): \(\mathbf a\cdot\mathbf b\), \(\mathbf a\,{}^{\perp}\!\cdot\,\mathbf b\) 계산, \(\varphi\) 추정.
  2. 정점 \( (0,0)\to(3,0)\to(3,2)\to(0,2) \) 직사각형의 부호가 있는 면적을 \( \sum \mathbf v_i\,{}^{\perp}\!\cdot\,\mathbf v_{i+1} \)로 구하라.
  3. \(\mathbf r=(2,1), \mathbf F=(-1,4)\): \(\tau=\mathbf r\,{}^{\perp}\!\cdot\,\mathbf F\)의 부호 해석.
  4. \(\mathbf a\)를 45° 회전한 \(\mathbf b\): \(\sin\varphi\)를 \( \dfrac{\mathbf a\,{}^{\perp}\!\cdot\,\mathbf b}{\|\mathbf a\|\,\|\mathbf b\|} \)로 확인.
FAQ

9) 자주 묻는 질문

▶ 2D에서도 외적 벡터가 꼭 필요하지 않나요?

필요하지 않다. 방향은 부호로, 크기는 면적/토크 크기로 충분하다.

▶ 3D 문제로 확장하면?

3D에선 외적을 사용한다. 2D에 국한된 상황에서만 \( {}^{\perp}\!\cdot \) 접근이 단순하고 안전하다.
