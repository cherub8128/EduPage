회전하는 사각 물통의 포물선 단면

논문 기반 수학·물리 탐구 교재
📚 개요 🎮 시뮬레이션 📐 수식·심화

학습 목표 이 교재에서 배우는 것

실생활 현상(회전하는 물통)을 관찰하고, 논문 기반 실험을 바탕으로 수학적 모델링과 컴퓨터 시뮬레이션을 통해 이해하는 탐구 중심 학습입니다.

수학·물리 통합 논문 기반 학습 컴퓨터 시뮬레이션
  • 관찰: 회전하는 물통의 자유수면이 포물선이 되는 현상을 이해
  • 모델링: 힘의 평형, 압력, 부피 보존 등의 물리 법칙으로 수식화
  • 시뮬레이션: 이론 포물선 + 물리엔진 입자 유체를 동시에 비교
  • 심화: 무차원화, 다른 기하학적 형태(원통, 정육면체)로의 확장
  • 탐구: 실험 설계, 데이터 수집, 수학적 분석까지 직접 수행

STEP 1 논문 기반 현상 관찰

두 논문을 통해 "회전하는 용기의 자유수면은 포물선이 된다"는 현상을 탐구합니다.

📄 참고 논문 1: Monteiro et al. (2019)

"The Parabola of Revolution: An Experimental Study Using a Rectangular Container"
직사각형 좁은 물통을 회전시키며, 스마트폰 자이로스코프와 비디오 분석을 활용해 자유수면이 이론 포물선과 일치함을 검증한 교육용 실험 논문입니다.

📄 참고 논문 2: Menker & Herczyński (2022)

"Dry Patches and Rotating Containers"
다양한 기하학적 형태(원통, 정육면체 등)의 용기에서 임계 각속도 이상으로 회전할 때 중심에 "마른 영역(dry patch)"이 생기는 현상을 실험·이론적으로 분석한 논문입니다.

핵심 관찰
  • 용기 모양(직사각형, 원통 등)과 상관없이 자유수면 단면이 포물선
  • 각속도 \(\omega\) 가 커질수록 포물선이 가파르게 휘어짐
  • 임계 각속도 이상에서는 중심이 바닥에 닿으며 "마른 영역" 형성
  • 실험 데이터를 통해 중력가속도 \(g\) 를 역추정할 수 있음
[물리 법칙에서 수식 유도] 회전 좌표계에서 유체 요소에 작용하는 힘: - 중력: \(\rho \mathbf{g} = -\rho g\,\hat{z}\) - 원심력: \(\rho \mathbf{a} = \rho \omega^2 r\,\hat{r}\) 정수압 평형 (정상 상태): \[ \nabla p = \rho(\mathbf{g} - \mathbf{a}) \] 이를 적분하면 자유수면에서: \[ z_0(r) = \frac{\omega^2}{2g} r^2 + C \] → 포물선 형상을 수학적으로 확정

STEP 2 수학적 모델링

복잡한 유체역학 방정식을 단순화된 가정으로 축약하여 시뮬레이션 가능한 모델을 만듭니다.

[단순화 가정] (1) 정상 상태(steady): \[ \frac{\partial}{\partial t} = 0 \] (2) 강체 회전: \[ \mathbf{u} = \omega r\,\hat{\theta} \] (3) 점성 무시: 전단응력 무시, 압력장만 고려 (4) 직사각형 용기의 1차원 근사: 폭 \(L\), 깊이 \(d\), 한 방향으로만 변화 [직사각형 용기 자유수면] 부피 보존을 고려하면: \[ z_0(r) = H - \frac{\omega^2}{2g} \left(\frac{L^2}{12} - r^2\right) \] 여기서 \(-\frac{L}{2} \le r \le \frac{L}{2}\) [주요 물리량] 꼭짓점(중심) 높이: \[ z_v = H - \frac{\omega^2 L^2}{24 g} \] 임계 각속도 (꼭짓점이 바닥에 닿을 때): \[ \omega_c = \sqrt{\frac{24 g H}{L^2}} \]
모델 검증 전략

변증법적 접근

  • 정(正): 단순 포물선 이론식
  • 반(反): 물리엔진 입자 시뮬레이션 (이론과 다른 부분, 노이즈, 경계 효과)
  • 합(合): 두 결과를 비교하며 이론의 적용 범위 분석

검증 항목

  • \(\omega < \omega_c\): 이론과 시뮬레이션이 잘 일치하는가?
  • \(\omega \approx \omega_c\): 마른 영역이 형성되는가?
  • \(\omega > \omega_c\): 모델이 벗어나는 지점은 어디인가?
실험 설계 예시
  • 회전대 + 직사각형 물통 + 스마트폰(자이로)
  • 비디오 촬영 → Tracker 등으로 수면 높이 측정
  • 여러 \(\omega\) 에서 측정 → 회귀분석으로 \(g\) 추정

STEP 3 인터랙티브 시뮬레이션

이론 포물선과 물리엔진 입자 유체를 동시에 시각화하여 비교할 수 있습니다.

시뮬레이션 안내

시뮬레이션 페이지에서는 다음을 할 수 있습니다:

  • 슬라이더로 \(\omega, H, L, g\) 조절
  • 2D 포물선 단면 그래프 실시간 업데이트
  • 3D 시각화: 이론 포물선 수면(파란색) + Cannon.js 입자 유체(주황색)
  • 임계 각속도 \(\omega_c\) 계산 및 현재 상태 분석
