5. **심화(개별화)**

   * 논문에서 사용한 1차원 모델을 2차원 상황에 연결하고, 스스로 변인을 하나 추가하거나 문제를 재구성하여 **개별 탐구 주제**를 설계할 수 있다.

# 1. 논문 요약 (교사용)

> Peastrel, Lynch, Armenti Jr. (1980), *Terminal velocity of a shuttlecock in vertical fall*, American Journal of Physics 48(7).

### 1.1 논문이 한 일 (핵심 구조)

1. **문제 설정**

   * 셔틀콕을 수직으로 떨어뜨릴 때, 속도가 증가하다가 어느 순간 **종단속도(terminal velocity)**로 포화하는 현상을 관찰.

2. **모델 후보 3개**

   * (1) 공기저항 없음: (\dfrac{dv}{dt}=g)
   * (2) 속도에 비례하는 저항: (\dfrac{dv}{dt}=g - c_1 v)
   * (3) 속도의 제곱에 비례하는 저항: (\dfrac{dv}{dt}=g - c_2 v^2)

3. **실험**

   * 계단실에서 셔틀콕 수직 낙하를 고속 촬영.
   * 위치–시간 데이터를 얻고, 속도를 계산한 뒤 세 모델과 비교.

4. **결론**

   * 셔틀콕의 경우, 속도 제곱에 비례하는 저항 모델(3)이 데이터와 가장 잘 맞았다.
   * 이 모델에서 종단속도는
     [
     v_T = \sqrt{\frac{g}{c_2}}
     ]
     로 표현된다.

### 1.2 우리 수업에서의 활용 포인트

* 논문은 **1차원(수직)**에 집중하지만,
* 우리는 이 논리 구조를 가져와서

  * (1) **2차원 포물선(공기저항 무시)** → 기준 모델
  * (2) **2차원 셔틀콕 궤적** → 포물선에서 얼마나 어긋나는지 관찰
  * (3) **2차원 항력 모델**(제곱저항)로 업그레이드
* 즉, 논문은 “공기저항 모델 경쟁”의 **철학과 아이디어**를 제공하고, 실제 수업에서는 이를 2D로 확장하는 방식입니다.

### 2차시 – 2D 궤적 실험 (공: 기준 포물선)

* **학습 목표**

  * 공(작은 공/구슬)의 2D 투사 운동을 Tracker로 기록하고, 포물선 모델과 비교할 준비를 한다.

* **실험 설계 (공 통제군)**

  * **조작변인**

    * 발사각 (\theta) (예: 30°, 45°, 60° 중 한 가지 고정)
  * **종속변인**

    * (x(t), y(t)), 궤적 (y(x))
  * **통제변인**

    * 물체 종류(작은 공)
    * 발사 속도(경사 레일 + 테이블 끝 구조로 최대한 일정하게)
    * 카메라 위치/각도, fps (가능하면 60fps, 삼각대)
    * 스케일(바닥에 격자/테이프)
    * 실내 바람 최소화

* **실험 절차**

  1. 경사 레일(또는 긴 판)을 테이블에 올려 공을 굴리다가 테이블 끝에서 수평 혹은 일정 각도로 발사.
  2. 카메라는 **운동 평면에 정면**, 멀리서 촬영.
  3. Tracker에서:

     * 스케일 설정(바닥 테이프 또는 벽의 줄자)
     * 좌표계 설정: (x) 수평, (y) 수직(위 + 또는 아래 + 일관되게)
     * 공의 중심점 추적 → (x(t), y(t)) 데이터 얻기.

* **교사 유의점**

  * 공기저항이 상대적으로 적으므로, 무저항 포물선 모델이 어느 정도 잘 맞아야 한다.
  * **발사 조건의 재현성**이 중요. 같은 실험 여러 번 반복해서 산포도 보는 것도 좋다.

### 3차시 – 2D 궤적 실험 (셔틀콕: 실험군)

* **학습 목표**

  * **종속변인**

    * (x(t), y(t)), (v_x(t), v_y(t)), 속력 (v(t))
  * **통제변인**

    * 발사 위치/각도(튜브 발사기 등으로 고정)

  * PVC 파이프나 셔틀 튜브를 일정 각도로 고정.

  * 초기 조건은 완전히 같지 않아도, **초기 0.1~0.2초 데이터로부터 (v_{0x}, v_{0y})를 추정**해 보정할 계획임을 학생에게 안내.

* **활동**

  1. 셔틀 발사/촬영.

     * 스케일, 좌표계 설정.
     * 코르크 중심점 추적.

  * 이 어긋남을 “모델이 틀렸다 → 모델을 고쳐야 한다”는 논리로 연결할 준비를 한다.

### 4차시 – 단순화(일반화): 이상적인 포물선 모델 정리 및 공 데이터 검증

* **학습 목표**

  * 무저항 가정 하에서 2D 투사체 운동이 포물선임을 유도하고, 공 데이터에 이 모델을 피팅하여 오차를 확인한다.

* **수학적 유도 (교사용)**

  가정:

  * 공기저항 없음
  * 중력가속도 (\vec g = (0,-g)) 일정
  * 운동은 2D, 질점 모델

  뉴턴의 운동법칙:
  [
  m\frac{d^2 x}{dt^2} = 0, \quad
  m\frac{d^2 y}{dt^2} = -mg
  ]

  통합하면
  [
  \frac{d^2 x}{dt^2}=0 \Rightarrow \frac{dx}{dt}=v_{0x} \Rightarrow x(t)=x_0+v_{0x}t
  ]
  [
  \frac{d^2 y}{dt^2}=-g \Rightarrow \frac{dy}{dt}=v_{0y}-gt \Rightarrow y(t)=y_0+v_{0y}t-\frac12gt^2
  ]

  시간 (t)를 소거하면:
  [
  t = \frac{x-x_0}{v_{0x}},
  ]
  [
  y(x) = y_0 + v_{0y}\frac{x-x_0}{v_{0x}} - \frac12 g\left(\frac{x-x_0}{v_{0x}}\right)^2
  = -\frac{g}{2 v_{0x}^2}(x-x_0)^2 + \frac{v_{0y}}{v_{0x}}(x-x_0) + y_0
  ]
  → **이차함수(포물선)**

* **학생 활동**

  1. 공 데이터의 (x(t))를 직선으로 피팅 → (v_{0x}) 추정.
  2. (y(t))를 이차함수로 피팅 → (v_{0y}, g) 추정.
  3. 포물선 (y(x))를 만들어서 실측 점과 겹쳐 그림.
  4. 시간별 오차 (|y_{\text{실측}}(t)-y_{\text{모델}}(t)|) 그래프 그려보기.

* **교사 유의점**

  * “공기저항이 0은 아니지만, 공 데이터에서는 모델이 꽤 잘 맞는다”는 메시지를 강조.
  * 포물선 모델의 **정당성을 확보**한 뒤, 셔틀에서의 실패를 다루어야 “정–반–합” 구조가 명확하다.

### 5차시 – 셔틀콕 2D 항력 모델 & 시뮬레이션

* **학습 목표**

  * 셔틀콕에서 공기저항을 고려한 2D 모델을 세우고, 수치적으로 시뮬레이션한다.
  * 데이터에 맞는 항력 계수를 찾기 위한 기본 아이디어를 익힌다.

#### 5.1 2D 제곱항력 모델 설정

* 속도 (\vec v=(v_x, v_y)), 속력 (v=\sqrt{v_x^2+v_y^2}).
* 중력: (\vec g=(0,-g)).
* 항력: 방향은 (-\vec v), 크기는 (k v^2) (간단 모델):
  [
  \vec F_d = - m k v \vec v
  ]
* 뉴턴의 법칙:
  [
  m\frac{d\vec v}{dt} = m\vec g + \vec F_d
  ]
  따라서
  [
  \frac{d\vec v}{dt} = \vec g - k v \vec v
  ]

성분으로 쓰면 (위가 + 일 때):
[
\frac{dv_x}{dt} = -k v v_x,\qquad
\frac{dv_y}{dt} = -g - k v v_y
]
[
\frac{dx}{dt} = v_x,\qquad
\frac{dy}{dt} = v_y
]

> **포인트**: 이 미분방정식은 고2 수준에서 “해석적으로 풀기 어렵다”고 정직하게 말하고, **수치적분(오일러)**으로 접근하게 한다.

#### 5.2 Euler 방법에 의한 수치 해 (교사용 설명)

시간 간격 (\Delta t)에 대해:
[
v_{x,n+1} = v_{x,n} + \left(-k v_n v_{x,n}\right)\Delta t
]
[
v_{y,n+1} = v_{y,n} + \left(-g - k v_n v_{y,n}\right)\Delta t
]
[
x_{n+1} = x_n + v_{x,n+1}\Delta t,\quad
y_{n+1} = y_n + v_{y,n+1}\Delta t
]
여기서
[
v_n = \sqrt{v_{x,n}^2 + v_{y,n}^2}
]

#### 5.3 GeoGebra 시뮬레이션(예시 절차)

1. 파라미터 슬라이더 만들기:

   * (g, k, \Delta t, v_{0x}, v_{0y})
2. 초기값 정의:

   * (x_0, y_0)
3. 수열 정의:

   * ((x_n, y_n, v_{x,n}, v_{y,n}))을 점화식으로 정의 (Sequence 기능)
4. 점 리스트를 궤적으로 연결해서 그림.
5. 셔틀 데이터의 실측 점과 겹쳐서 비교.

#### 5.4 HTML+JS 구현(개략 코드)

```html
<script>
const g = 9.8;
let k = 0.2;        // 항력 계수 (학생이 조절)
let dt = 0.01;

function simulate(v0x, v0y, steps){
  let x = 0, y = 0;
  let vx = v0x, vy = v0y;
  const traj = [];
  for(let i=0;i<steps;i++){
    const v = Math.hypot(vx, vy);
    const ax = -k * v * vx;
    const ay = -g - k * v * vy;
    vx += ax * dt;
    vy += ay * dt;
    x += vx * dt;
    y += vy * dt;
    traj.push({x, y});
    if(y < 0) break; // 지면에 닿으면 종료
  }
  return traj;
}
</script>
```

이 궤적을 `<canvas>`나 SVG에 그리면 된다.

### 6차시 – 모델 검증 & 심화 주제(개별화)

* **학습 목표**

  * 시뮬레이션 결과와 실측 데이터를 정량적으로 비교, (k)를 조정해 모델 적합도를 높이는 과정을 경험한다.
  * 개인별/팀별로 “한 줄 더” 모델을 고치거나 문제를 재정의하여 심화 탐구 주제를 만든다.

* **활동**

  1. **초기조건 추정**

     * 실측 데이터를 이용해 첫 2~3프레임에서 (v_{0x}, v_{0y}) 근사.

  2. **k 스캔**

     * 여러 (k)값(예: 0.05 ~ 0.6)을 돌려서 각각에 대해 궤적 계산.
     * 실측 점들과의 거리 제곱합
       [
       E(k)=\sum_i\left[(x_i - x^{(sim)}_i(k))^2 + (y_i - y^{(sim)}_i(k))^2\right]
       ]
       을 구하고, 가장 작은 (k)를 선택.

  3. **모델 비교**

     * 무저항 모델 vs 항력 모델의 오차 그래프 비교.
     * “어떤 모델이 더 설득력 있는지” 이유를 글로 설명.

  4. **심화 주제(예시)**

     * 스핀(회전)을 추가해서 양력 항을 넣어보는 모델.
     * 항력을 속도에 비례 + 속도제곱에 비례 혼합 형태로 두고 비율 조정.
     * 항력 있을 때 “최대 사거리 각도”가 45°가 아닌 것을 시뮬레이션으로 확인.
     * 바람(상수 바람)을 도입해 (\vec v - \vec w) 형태로 수정.

* **평가**

  * 실험 설계·데이터 처리·모델링 과정의 논리성 + 심화 주제의 창의성과 설득력으로 종합 평가.

# 3. 수학 유도 상세 (교사용, 심화용)

## 3.1 1D 제곱항력 모델(논문과 연결)

논문에서 사용한 1D 모델(아래 방향 +)은
[
\frac{dv}{dt} = g - k v^2
]
입니다. 여기서 (k>0)는 항력 계수, (v)는 아래 방향 속도입니다.

### 3.1.1 변수 분리

[
\frac{dv}{dt} = g - k v^2
\Rightarrow
\frac{dv}{g - k v^2} = dt
]

종단속도 (v_T)를
[
v_T = \sqrt{\frac{g}{k}}
]
로 두면
[
g - k v^2 = g \left(1 - \frac{v^2}{v_T^2}\right)
]
따라서
[
\int \frac{dv}{1 - (v^2/v_T^2)} = \frac{g}{v_T} \int dt
]

좌변 적분:
[
\int \frac{dv}{1 - (v^2/v_T^2)}
= \int \frac{(1/v_T),du}{1-u^2} \quad(u = v/v_T)
= \frac{1}{v_T} \operatorname{artanh}(u)
]
따라서
[
\operatorname{artanh}\left(\frac{v}{v_T}\right) = \frac{g}{v_T} t + C
]

초기조건 (v(0)=0)을 대입하면 (C=0),
[
\operatorname{artanh}\left(\frac{v}{v_T}\right) = \frac{g}{v_T} t
\Rightarrow
\frac{v}{v_T} = \tanh\left(\frac{g}{v_T}t\right)
]
[
v(t) = v_T \tanh\left(\frac{g}{v_T}t\right)
]

> 여기서 (\tanh)는 쌍곡선 탄젠트 함수로, 고2에게는 “새로운 함수”로 소개하거나, 수열/지수함수 심화에서 간단히 맛만 보는 정도로 사용.

### 3.1.2 위치 (y(t)) 유도

[
\frac{dy}{dt}=v(t)
= v_T \tanh\left(\frac{g}{v_T}t\right)
]

적분하면,
[
y(t) = \int_0^t v_T \tanh\left(\frac{g}{v_T}s\right),ds
= \frac{v_T^2}{g} \ln \cosh\left(\frac{g}{v_T}t\right)
]
(정확한 적분은 교사용 심화용; 학생에게는 “이와 같은 형태의 해가 나온다” 정도로 소개해도 충분)

이 1D 해는 논문에서 종단속도와 궤적을 나타내는 식과 구조가 같습니다.

---

## 3.2 2D 항력 모델: 왜 해석해가 어려운지

2D에서는
[
\frac{d\vec v}{dt} = \vec g - k v \vec v,\quad v=|\vec v|
]
가 되어 (v)가 (\vec v)의 크기에 의존하므로, (v_x,v_y)에 대한 coupled ODE(연립 미분방정식)가 됩니다.

[
\frac{dv_x}{dt} = -k v v_x, \quad
\frac{dv_y}{dt} = -g - k v v_y
]

이 식은

* 비선형이고,
* 두 성분이 서로 얽혀 있어서,

고등학교 수준에서는 **해석적으로 풀기 어렵다 → 수치적으로 푼다**는 메시지를 전달하기에 아주 좋은 예시입니다.

### 5. 셔틀콕 2D 모델링 & 시뮬레이션

#### 5.1 왜 포물선이 깨질까?

셔틀콕은

* 모양이 복잡하고,
* 공기저항이 매우 크며,
* 이동 방향과 반대 방향으로 강한 힘을 받는다.

공기저항을 단순하게 “속도의 제곱에 비례”한다고 가정해 보자.

#### 5.2 2D 제곱항력 모델(개념)

속도 (\vec v=(v_x, v_y)), 속력 (v=\sqrt{v_x^2+v_y^2}).
중력 (\vec g=(0,-g)).

**가정**

* 항력 방향: (-\vec v)
* 크기: (k v^2)

그러면
[
\frac{d\vec v}{dt} = \vec g - k v \vec v
]

성분으로 쓰면:
[
\frac{dv_x}{dt} = -k v v_x,\qquad
\frac{dv_y}{dt} = -g -k v v_y
]
[
\frac{dx}{dt} = v_x,\qquad
\frac{dy}{dt} = v_y
]

#### 5.3 수치적분(Euler 방법, 아이디어)

시간을 (\Delta t) 간격으로 나눈다.

* 속도 업데이트:
  [
  v_{x,n+1} = v_{x,n} + a_{x,n} \Delta t,\quad
  v_{y,n+1} = v_{y,n} + a_{y,n} \Delta t
  ]
  여기서
  [
  a_{x,n} = -k v_n v_{x,n},\quad
  a_{y,n} = -g -k v_n v_{y,n}
  ]
  [
  v_n = \sqrt{v_{x,n}^2 + v_{y,n}^2}
  ]

* 위치 업데이트:
  [
  x_{n+1} = x_n + v_{x,n+1}\Delta t,\quad
  y_{n+1} = y_n + v_{y,n+1}\Delta t
  ]

이 과정을 컴퓨터(또는 지오지브라의 수열)로 반복하면 궤적을 구할 수 있다.

#### 5.4 시뮬레이션 과제

1. 공기저항 **없음** 모델로 궤적을 시뮬레이션해서 공의 데이터와 비교.
2. **제곱항력 모델**로 궤적을 시뮬레이션해서 셔틀 데이터와 비교.
3. 항력 계수 (k)를 여러 값으로 바꾸어 가며,
   실측 궤적과 가장 비슷해 보이는 (k)를 찾아보기.
