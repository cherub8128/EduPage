Windows LaTeX 협업 환경 구축

MiKTeX, Visual Studio Code, Live Share를 활용하여 실시간으로 LaTeX 문서를 공동 편집할 수 있는 완벽한 환경을 설정하는 가이드입니다.

Step 1: MiKTeX 설치하기

MiKTeX는 LaTeX 문서를 PDF로 변환(컴파일)하는 데 필요한 모든 핵심 프로그램을 제공하는 '배포판'입니다. 패키지 자동 설치 기능이 가장 큰 장점입니다.

  1. 다운로드: 공식 사이트에서 설치 프로그램을 받으세요.
  2. 핵심 설정: 설치 중 'Install missing packages on-the-fly'를 'Yes'로 변경하여 필요한 패키지를 컴파일 시 자동으로 설치하도록 설정하세요.

Step 2: VS Code & LaTeX Workshop

VS Code에 'LaTeX Workshop' 확장 프로그램을 설치하면, 코드 작성부터 실시간 미리보기, 컴파일까지 모든 작업을 한 곳에서 해결할 수 있습니다.

VS Code 확장 프로그램 탭(Ctrl+Shift+X)에서 LaTeX Workshop을 검색하여 설치하세요.

Step 3: XeTeX 기본 컴파일러 설정

한글을 원활하게 사용하고 다양한 폰트를 쉽게 적용하기 위해 XeTeX 컴파일러를 LaTeX Workshop의 기본값으로 설정합니다.

Ctrl+Shift+P로 VS Code의 설정 파일(settings.json)을 열고 아래 코드를 추가하세요.

"latex-workshop.latex.tools": [
    {
        "name": "xelatex",
        "command": "xelatex",
        "args": [
            "-synctex=1",
            "-interaction=nonstopmode",
            "-file-line-error",
            "%DOCFILE%"
        ]
    }
],
"latex-workshop.latex.recipes": [
    {
        "name": "xelatex",
        "tools": ["xelatex"]
    }
],
"latex-workshop.latex.recipe.default": "xelatex"

Step 4: 한글(KoTeX) 예제 만들기

example.tex 파일을 만들고 아래 코드를 붙여넣은 후, 빌드(Ctrl+Alt+B)하여 한글 PDF가 잘 생성되는지 확인해 보세요.

% !TEX program = xelatex
\documentclass[12pt]{article}
\usepackage{amsmath}
\usepackage{kotex}

% --- 폰트 설정 (사용자 환경에 맞는 폰트 필요) ---
\usepackage{fontspec}
\setmainfont{Noto Serif KR}
\setsansfont{Noto Sans KR}
\setmonofont{D2Coding}

\title{수학 논문 예시}
\author{차형준}
\date{\today}

\begin{document}
\maketitle

\section{서론}
LaTeX는 복잡한 수식을 간결하고 아름답게 표현하는 데 매우 강력한 도구입니다. 예를 들어, 아인슈타인의 유명한 질량-에너지 등가 원리인 $E=mc^2$와 같은 인라인 수식도 쉽게 작성할 수 있습니다.

\end{document}

Step 5: Live Share로 공동 작업

VS Code의 Live Share 확장 프로그램을 설치하고 로그인하면, 초대 링크 하나로 다른 사람과 실시간으로 문서를 함께 편집할 수 있습니다.

핵심 기능: 호스트만 LaTeX 컴파일 환경이 구축되어 있으면 됩니다. 참여자는 LaTeX 설치 없이도 공동 편집 및 PDF 결과 확인이 가능하여 협업이 매우 간편해집니다.
