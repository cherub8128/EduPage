KaTeX 사용법 (Cheatsheet)

KaTeX는 웹에서 빠르고 쉽게 수학 수식을 렌더링하는 라이브러리입니다. 아래는 기본적인 사용법입니다.

기본 설정

HTML의 <head> 태그 안에 아래의 CDN 링크를 추가하여 KaTeX를 페이지에 포함시킵니다.

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.10/dist/katex.min.css">

<!-- JS -->
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.16.10/dist/katex.min.js"></script>

<!-- Auto-render Extension -->
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.16.10/dist/contrib/auto-render.min.js"></script>

그리고 <body> 태그 마지막에 아래 스크립트를 추가하여 페이지 내의 수식을 자동으로 렌더링하도록 설정합니다.

<script>
    document.addEventListener("DOMContentLoaded", function() {
        renderMathInElement(document.body, {
            delimiters: [
                {left: "$$", right: "$$", display: true},
                {left: "$", right: "$", display: false},
                {left: "\[", right: "\]", display: true},
                {left: "\\(", right: "\\)", display: false}
            ]
        });
    });
</script>

사용 예제

인라인 수식 (Inline)

문장 안에 수식을 넣으려면 $ 기호로 감쌉니다.

피타고라스 정리는 $a^2 + b^2 = c^2$ 입니다.

결과: 피타고라스 정리는 $a^2 + b^2 = c^2$ 입니다.

블록 수식 (Display/Block)

수식을 별도의 줄에 크게 표시하려면 $$ 기호로 감쌉니다.

근의 공식은 다음과 같습니다. $$x = {-b pm s t r o t} { b ^ 2 - 4 a c } { 2 a }$$

결과: 근의 공식은 다음과 같습니다. $$x = {-b pm s t r o t} { b ^ 2 - 4 a c } { 2 a }$$

분수와 제곱근

$$\ f r a c t { 1 } { 2 }   \ s q r t { x ^ 2 + y ^ 2 }$$

결과:

$$\ f r a c t { 1 } { 2 } \ s q r t { x ^ 2 + y ^ 2 }$$

그리스 문자

$\ a l p h a, \ b e t a, \ g a m m a, \ D e l t a, \ O m e g a$

결과: $\ a l p h a, \ b e t a, \ g a m m a, \ D e l t a, \ O m e g a$

합계 및 적분

$$\ s u m _ { i = 1 } ^ { n }   i ^ 2   =   \ f r a c t { n ( n + 1 ) ( 2 n + 1 ) } { 6 }   \   \ i n t g _ { a } ^ { b }   f ( x ) d x$$

결과:

$$\ s u m _ { i = 1 } ^ { n } i ^ 2 = \ f r a c t { n ( n + 1 ) ( 2 n + 1 ) } { 6 } \ \ i n t g _ { a } ^ { b } f ( x ) d x$$

행렬

$$ 
A = \ b e g i n { c a t r i x } 
 a   &   b \ \ 
 c   &   d \ \ 
 \ e n d { c a t r i x } 
$$

결과:

$$ A = \ b e g i n { c a t r i x } a & b \ \ c & d \ \ \ e n d { c a t r i x } $$
